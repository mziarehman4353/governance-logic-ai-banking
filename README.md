
# Governance Logic Analysis in Islamic and Western Banking using NLP and LLMs

## Overview

This project presents an AI-driven comparative governance analysis of Islamic and Western banking systems using Natural Language Processing (NLP), Machine Learning, and Large Language Models (LLMs).

The study introduces a novel Governance Logic Index for identifying technocratic, ethical, and hybrid governance patterns in financial texts.

---

## Features

- NLP preprocessing pipeline
- TF-IDF vectorization
- Keyword intelligence analysis
- KMeans clustering
- PCA visualization
- Governance Logic Index
- Statistical hypothesis testing
- Logistic Regression validation
- FLAN-T5 prompt engineering
- Zero-shot classification baseline
- McNemar significance testing
